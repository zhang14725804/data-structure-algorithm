### 二分的流程

    (1)确定边界
    (2)代码框架
    (3)设置check
    (4)判断区间如何更新
    (5)注意mid取值

### 模板1(下取整)

```bash
func bsearch1(l,r int){
    while(l<r){
        mid := (l+r) >> 1
        if check(mid){
            r = mid
        }else{
            l = mid+1
        }
    }
    return l
}
```

### 模板2(上取整)

```bash
func bsearch2(l,r int){
    while(l<r){
        mid := (l+r+1) >> 1
        if check(mid){
            l = mid
        }else{
            r = mid-1
        }
    }
    return l
}
```
